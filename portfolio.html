Good Times Local Loans

Live Website Link

Good Times Local Loans is a two sided market place that allows users to browse locally listed items which they can hire for an agreed period of time. Users can also list their own items in exchange to cash.

- Functionality / features
  • Image uploading for item listings.
  • Images are stored on cloudinary server.
  • Authentication throughout the site which is implemented using the devise gem.
  • Roles are set to the users and scoped to individual items by using the gem rolify.
  • Secure tansactions between borrowers and lenders whicn is achievable through Stripe.

The Curious Case of Dudley Manor

Github Link

'The Curious Case of Dudley Manor' is a text-based investigation and survival horror game developed in Ruby. It is inspired by the Goosebumps novels, chooses your own adventures and the 2010 game Amnesia: The Dark Descent. The player has been assigned there very first case as a detective to discover the mysterious disappearance of the Dudley family. The player is free to explore the Dudley manor in search of clues using text input. As the player descends deeper into the manor clues will be revealed but the house will take on a sinister vibe. The game Amnesia is a good reference. Keys can be found to unlock new rooms and items can be found to help the player fight back against ghastly creatures that lurk in Dudley manor. The game will provide a challenging but narrative focussed experience. The player’s progress can be saved/loaded, the data which will be stored as a JSON file.
